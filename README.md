# AUDIO-PLAYER
A simple desktop audio player built with Python, Tkinter, and Pygame. This project was created as a hands-on learning experience to understand GUI development, audio playback, and file handling in Python.

Features
Play audio files
Pause and resume playback
Stop music
Quit the application
Automatically loads songs from a selected folder
Displays available songs in the interface
Simple and user-friendly graphical interface
Technologies Used
Python
Tkinter
Pygame Mixer
OS Module

Why I Built This

I built this project while learning Python to move beyond tutorials and create something practical. The goal was to understand how real applications work by combining graphical interfaces, event handling, file management, and audio playback into a single project.

What I Learned

Working with external libraries such as Pygame
Handling button events and user interactions
